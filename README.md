## Кластер нод с генератором

> Приложение, экземпляры которого способны создавать согласованный кластер. Один из экземпляров раз в 1 секунду генерирует сообщение для другого случайного экземпляра в кластере.

## Запуск

```bash
$ git clone https://github.com/qrinef/nodejs-instances.git
$ docker-compose up -d
```

## Журнал работы

> По умолчанию, запускается 5 экземпляров. Чтобы лучше понимать что происходит, необходимо смотреть на все одновременно, т.е. открыть 5 консолей. Или воспользоваться IDE для удобства. 

```bash
$ docker-compose logs -f instance-1
$ docker-compose logs -f instance-2
$ docker-compose logs -f instance-3
$ docker-compose logs -f instance-4
$ docker-compose logs -f instance-5
```

## Завершение экземпляра

> При завершении любого экземпляра, назначается новый генератор сообщений. Сопровождается уведомлением в журнале работы.

> Необходимо выбрать любой экземпляр и завершить его. Завершать можно как генератора, так и слушателей.

```bash
$ docker-compose stop instance-1
```
